财务会计的主要特征： 

①以货币作为主要计量尺度； 

②会计记录要有真凭实据； 

③会计反映应当连续、系统、全面和综合； 

④会计的基本职能是反映与控制； 

⑤需要运用专门的程序和方法； 

⑥会计的本质是一种价值管理活动。 

1. 财务会计的特点： 

①作为一个经济系统，主要由确认、计量、记录 和报告等一系列元素（或环节）构成； 

②提供以财务报表为核心的信息； 

③主要提供货币量化的历史信息； 

④主要向利益相关者（外、内）提供信息 

⑤符合会计准则和相关会计制度规定。 

.会计信息的使用者和提供者： 

投资者，债权人，政府机关，供应商，客户，媒体、中介、师生等其他使用者；企业内部管理层和员工。 

会计信息的质量特征： （要知道特征并会背特征后面的那句话，可能考简答） 

①可靠性：企业应当以实际发生的交易或事项为依据进行会计确认、计量和报告，如实反映符合确认和计量 要求的各项会计要素及其相关信息，保证会计信息真实可靠，内容完整。 

②相关性：相关性是指提供的会计信息应当与财务报告者的经济决策需要相关，有助于财务会计报告使用者 对企业过去、现在或者未来的情况作出评价或者预测。 

③可理解性：企业提供的会计信息应当清晰明了，便于会计信息使用者理解和使用。 

④可比性：企业提供的会计信息应当口径一致，相互可比。 

⑤实质重于形式：是指企业应当按照交易或事项的经济实质进行会计确认、计量和报告， 不应当以交易或者事项的法律形式为依据。 

⑥重要性：企业提供的信息应当反映与企业财务状况、经营成果和现金流量等有关的所有重要交易或者事项。 依据：成本效益原则 标准：数量和性质 

⑦谨慎性：亦称稳健性原则或保守性原则，是指企业对交易或者事项进行会计确认、计量和报告时应当保持 应有的谨慎，不应高估资产或收益、少计负债或费用。 

⑧及时性：指企业对于已经发生的交易或者事项，应当及时进行会计确认、计量和报告，不得提前或者延后。

①资产： 资产是指企业过去的交易或者事项形成的、由企业拥有或者控制的、预期会给企业带来经济利益的 资源。 

②负债： 负债是指企业过去的交易或者事项形成的、预期会导致经济利益流出企业的现时义务。 

③所有者权益： 所有者权益是指企业资产扣除负债后由所有者享有的剩余权益，即净资产。 

④收入： 收入是指企业在日常活动中形成的、会导致所有者权益增加的、与所有者投入资本无关的经济利益 的总流入。 

PS：收入与利得的区别： 

收入是企业在销售商品、提供劳务、他人使用本企业资产等日常活动中形成的经济利益的总流入。 

利得通常从偶发的经济业务中取得，属于那种不经过经营过程就能取得或不曾期望获得的收益。利得可分为 直接计入当期损益的利得和损失与直接计入所有者权益的利得。 

⑤费用：费用是指企业在日常活动中发生的、会导致所有者权益减 少的、与向所有者分配利润无关的经济 利益的总流出。 

PS：支出、费用、成本与损失的辨析： 

支出：多指货币资金的流出； 

费用： 企业为销售商品、提供劳务等日常经营活动所发生的经济利益的流出，会导致所有者权益减少。一定 会计期间的费用与收入配比出现。 

成本：企业为生产产品、提供劳务而发生的各种耗费；成本通常与产品相联系，是在特定对象上为特定目的 而发生的耗费，因此被称为“对象化”的耗费。 

损失：由非日常经营活动所导致的经济利益的流出，会导致所有者权益减少；一定会计期间的损失与利得源 自不同的交易，因此不存在配比关系 

⑥利润：利润是指企业在一定会计期间的经营成果。

# 坏账损失：

1、债务人被依法宣告破产、撤销，其剩余财产确实不足清偿的应收账款； 

2、债务人死亡或依法被宣告死亡、失踪，其财产或遗产确实不足清偿的应收账款； 

3、债务人遭遇重大自然灾害或意外事故损失巨大，以致其财产（包括保险赔款等）确实无法清偿的应收账款； 

4、债务人逾期未履行偿债义务，经法院裁决，确实无法偿还的应收账款； 

5、逾期三年以上仍未回收的应收账款； 

6、经国家税务局应准核销的应收账款。

⑥存货： 企业在日常活动中持有以备出售的产成品或商品、处在生产过程中的在产品、在生产过程或提供劳务 过程中耗用的材料和物料等。存货成本包括采购成本、加工成本和其他成本。 

# 存货的特征：

1.存货是一种具有物质实体的有形资产； 

2.存货属于流动资产，具有较大的流动性； 

3.存货持有的最终目的是在正常生产经营过程中被出售，不论是可供直接出售，还是进一步加工后被出售；（不 包括为建造固定资产而购入的材料和用于产品生产的设备。） 

4.存货属于非货币性资产，存在价值减损的可能性。 

发出存货的计价方法： （判断和简答考的可能性大，要懂得计算） 

1. 个别计价法：逐一辨认各批发出存货和期末存货所属的购进或生产批次，分别按其购入或生产时所确定的 单位成本计算各批发出存货和期末存货成本。 

特点：存货的实物流转与成本流转一致；能准确地计算出发出存货成本和期末存货成本；操作量繁重。适 用于不能替代的或特定项目购入存货。 

2. 先进先出法：先入库的存货成本在后入库的存货成本之前转出，据此确定发出存货和期末存货的成本。 特点：期末存货成本比较接近现行的市场价值；物价上涨时，会高估企业当期利润和库存存货的价值；工 作量较大。 

3. 月末一次加权平均法：以本月全部收货成本加上月初存货成本，除以本月全部收货数量加上月初存货的数 量，据以计算出存货的加权平均单位成本，从而确定发出存货的成本和期末存货成本。 

特点：月末一次计算加权平均单位，比较简单，对存货成本的分摊较为折中；但是无法在账面上提供发出 和结算存货的单价和金额，不利于企业存货管理。 

4. 移动平均法：以每次收货成本加上原有存货成本，除以每次收货数量加上原有库存存货数量，据以计算出 加权平均单位成本，作为下次进货前计算各次发出存货成本依据。 

特点：能及时了解存货结存情况，比较客观地计算出发出和结存存货的成本；但计算工作量大，对收发频 繁企业不适用。

# 成本法优点：

（1）核算方法简便； 

（2）投资账户能够反映投资的成本； 

（3）企业实际获得的利润或现金股利情况能得到如实地反映，且与其现金流入在时间上基本吻合； 

（4）成本法确认的投资收益时间与我国税法上确认应纳税所得额时投资收益的确认时间一致，不存在会计 核算时间上与税法不一致的问题。 

（5）投资收益可以足额收回，避免了母公司在未收到子公司分红前垫付资金向投资者超额分配股利 

# 成本法缺点：

（1）投资账户只反映初始或追加投资时的投资成本，不能反映投资企业在被投资单位中的权益； 

（2）如果投资方有能力操纵被投资单位利润或股利的分配，则为投资方操纵利润提供了条件，这可能会导 致投资企业的投资收益不能真正反映其应当获得的利益。

实收资本： 投资者投入资本形成法定资本的价值。实收资本一般无需偿还，可长期周转使用。 

实收资本与投入资本的辨析： 

投入资本是投资者作为资本实际投入到企业的资金数额。一般情况下，投资者的投入资本构成企业的实收资本 （等于其登记认缴的注册资本）。但是在一些特殊情况下，投资者也会因种种原因超额投入（如溢价发行股票等）， 从而使其投入资本超过企业的注册资本，此时超过注册资本的部分应单独核算，计入“资本公积”。 

资本公积——资本｜股本溢价：投资者的出资额中超出其在企业注册资本或股本中所占份额的部分。 

.有限责任公司：（可能出现） 

初建有限责任公司时，各投资者按照合同、协议或公司章程投入企业的资本，应全部记入“实收 资本”账户， 企业的实收资本应等于企业的注册资本。增资时，新介入的投资者缴纳的出资额大于其按约定比例在注册资本 中所占的份额部分，不记入“实收资本”账户，而作为资本公积，记入“资本公积——资本溢价”账户。

24.财务报表：财务报表是对企业财务状况、经营成果和现金流量的结构性表述。财务报表至少应当包括资产负 债表、利润表、现金流量表、所有者权益变动表等四张报表，财务报表及其附注构成企业的财务报告。 （资产 负债表、利润表、现金流量表为重点） 

资产负债表： 资产负债表是报告会计主体在某一特定日期财务状况的报表。(记住它的依据和作用) 

依据：资产＝负债＋所有者权益（静态与时点） 

# 作用：

I．了解企业所拥有或控制的经济资源及其分布、企业资金的来源及其构成，有助于分析和评价企业的财务状况 和资本结构。 

II．了解企业资产和负债的数量关系及其流动性，有助于分析和评价企业的偿债能力和财务弹性。 

III．通过不同时期相同项目的比较，可以了解企业不同时点的财务状况情况，预测企业未来财务状况的发展趋 势。 

利润表： 利润表（损益表）是反映企业一定会计期间经营成果的报表。(记住依据和作用) 

依据：利润＝收入－费用＋利得－损失（动态与时期） 

# 作用：

I.可以为企业经营成果的分配提供重要依据； 

II.可以据以评价和考核企业的经营业绩； 

III.可以据以分析和评价企业的经营成果和获利能力； 

IV.可以据以分析和预测企业未来的现金流量； 

V.可以有助于企业管理当局进行经营决策。 

③现金流量表：现金流量表是反映企业在一定会计期间现金和现金等价物流入和流出的报表。（记住作用、现金 定义以及其包括什么、以及现金流量的三个活动） 

现金： 现金流量表中的现金是指企业的库存现金、可以随时用于支付的存款以及现金等价物。 

# 现金包括：

I、 库存现金： 指企业持有可随时用于支付的现金限额，与“库存现金”账户所包括的内容一致； 

II、 银行存款： 银行存款是指企业存在金融企业随时可以用于支付的存款； 

III、 其他货币资金： 是指企业存在金融企业有特定用途可随时用于支付的款项。与会计核算中的“其他货币资 金” 账户所包括的内容基本一致； 

IV、 现金等价物： 是指企业持有的期限短、流动性强、易于转换为已知金额现金、价值变动风险很小的投资。 期限短，一般是指从购买日起三个月内到期。现金等价物通常包括三个月内到期的短期债券投资等。 权益性投 资变现的金额通常不确定，因而不属于现金等价物。 

# 作用：

I.有助于分析和评价企业整体财务状况及其变动的原因； 

II.有助于分析和预测企业未来产生现金流量的能力； 

III.有助于了解和评价企业的支付能力和偿债能力； 

IV.有助于分析和揭示企业经营净收益与经营活动产生净现金流量之间差异的原因。 

# 现金流量表的三类活动：

.经营活动：经营活动是指企业投资和筹资活动以外的所有交易和事项。 

.投资活动： 投资活动是指企业长期资产的购建和不包括在现金等价物范围内的投资及其处置活动。 

.筹资活动：筹资活动是指导致企业资本及债务规模和构成发生变化的活动。

# D、杜邦分析体系

权益净利率=净利润÷股东权益 

$$
\begin{array}{l} = \frac {\text {净利润}}{\text {销售收入}} \times \frac {\text {销售收入}}{\text {总资产}} \times \frac {\text {总资产}}{\text {股东权益}} \\ = \text { 销售净利率 } \times \text { 总资产周转率 } \times \text { 权益乘数 } \\ \end{array}
$$

![](images/1a0241ad0e6df1716965784d9754a5b27563502aabb9acea9631965cfcfc186a.jpg)


杜邦分析体系是一个多层次的财务比率分解体系。各项财 务比率，在每个层次上与本企业历史或同业的财务比率比较， 比较之后向下一级分解，逐步覆盖企业经营活动的每一个环节 从而实现系统、全面评价企业经营成果和财务状况的目的。
