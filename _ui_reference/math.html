求导数 微积分
求函数 $f(x) = x^3 - 3x^2 + 2x - 1$ 的导数 $f'(x)$，并求其在 $x = 1$ 处的切线方程。
标准答案
1 求导
$f'(x) = 3x^2 - 6x + 2$
2 求 $x = 1$ 处的值
$f(1) = 1 - 3 + 2 - 1 = -1$
$f'(1) = 3 - 6 + 2 = -1$
3 切线方程
切线方程：$y - f(1) = f'(1)(x - 1)$
代入得：$y + 1 = -1(x - 1)$
即：$y = -x$
